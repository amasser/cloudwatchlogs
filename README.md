# cloudwatchlogs

Web UI for accessing AWS Cloud Watch Logs.

## Usage

```
Usage:
  cloudwatchlogs [OPTIONS]

Application Options:
      --host=
      --port=
      --auth-user=
      --auth-pass=
      --access-key=
      --secret-key=
      --region=
      --profile=

Help Options:
  -h, --help        Show this help message
```
